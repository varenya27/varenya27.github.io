Abstract

On our search for random website templates, we came across a latex themed css file and decided to add the theme to one the pages on our own website. We found that ripping off .css files from the internet activated the Nucleus Accumbens and decided to store some text (that isn't latin gibberish) for the time being. Additionally, we modified the code and added some personality to build some momentum towards creating something meaningful eventually. Whereas figuring out the best usage of this discovery is beyond the scope of our research, we hope this work inspires certain bored individuals to add a little pizzazz to their own personal websites.
